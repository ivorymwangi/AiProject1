# Ivory

A modern, minimalist skincare landing page inspired by the Rhode Beauty aesthetic. Built with **Bootstrap 5**, **HTML**, and **CSS**, featuring a warm grey & ivory palette, elegant typography (Fraunces + Inter), and a hand-crafted CSS bottle illustration.

## Preview

A single-section hero landing page with:

- Sticky top navigation
- Editorial split layout (headline + visual)
- Pure-CSS product bottle illustration
- Featured-in marquee
- Fully responsive Bootstrap grid




## File Structure


ivory/
├── index.html      # Markup + Bootstrap CDN links
├── styles.css      # Custom theme, layout, and bottle illustration
└── README.md

## Customization

- **Colors**: Edit the CSS custom properties at the top of `styles.css` (`--ivory`, `--grey-*`, etc.).
- **Typography**: Swap the Google Fonts `<link>` in `index.html` and update the `font-family` rules.
- **Copy**: All headline and button text lives in `index.html`.
